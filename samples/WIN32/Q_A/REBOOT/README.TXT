Sample:

Demonstration of the Win32 InitiateSystemShutdown API

Summary:

The REBOOT sample illustrates the use of the
InitiateSystemShutdown function to locally or remotely
trigger a system to shutdown (reboot).

More Information:

This sample also functions as a handy reboot utility.
The command syntax for the utility is as follows:

     REBOOT \\MachineName

where MachineName is name of the machine to reboot.
Use '.' for the MachineName, without the leading \\,
to reboot the local machine.
