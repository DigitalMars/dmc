 Sample: GetDeviceCaps Demonstration Program

Summary:

The GETDEV sample is a dialog box that displays the result
of the GetDeviceCaps call with all of the possible input
parameters. Six of the numeric results (TECHNOLOGY,
LINECAPS, POLYGONALCAPS, TEXTCAPS, CLIPCAPS, and RASTERCAPS)
are expanded to show the constant string from WINGDI.H.
