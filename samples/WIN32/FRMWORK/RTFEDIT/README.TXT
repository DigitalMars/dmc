PURPOSE:
    Show a very simple implementation of the RTF edit control.  See

    Based on Generic

COMMENTS:
    This is a 32-bit only sample.

MODULE MAP:
    Dispatch- Message dispatching routines
    WinMain - Calls initialization functions and processes the message loop
    Generic - Implements the windows procedure for the main application window
    Init    - Performs application and instance specific initialization
    About   - Defines a standard about dialog box.
    Misc    - Defines the applications specific commands not related to
                a specific module.
