complex, _Complex_I, imaginary, _Imaginary_I, I

Header

	complex.h

Description

complex
Is a macro expanding to _Complex.
_Complex_I
Is a macro expanding to a const expression of type _Complex float and has a value of 0+I.
imaginary, _Imaginary_I, I
Are all macros expanding to a const expression of type _Imaginary float and have a value of I.

Compatibility

Win32, ANSI C99 7.3.1
